# ghrd

A command to get GitHub Repo details(stars,forks,watchers) of current working directrory(should be a git directory)

## Installation

`npm install ghrd --global`

## How to Use

```
cd github-app

ghrd
# https://github.com/username/reponame

 Stars   : stars_count
 Forks   : forks_count
 Watchers: watchers_count
```
